Setting up a Virtual Environment and Running the Code

1. Navigate to the directory where the file finder code is located (`<filefinder directory>`).
2. Create a virtual environment using Python's `venv` module:  
   
       python -m venv venv
   
3.  Activate the virtual environment:
   
      .\venv\Scripts\activate
   
4. Install the required Python packages listed in `requirements.txt`:
  
      pip install -r requirements.txt
   
5.  Run the Python script `file_info_version_22.py`:
   
       python file_info_version_22.py

If the virtual machine does not run - 
If you encounter issues with running the virtual machine, ensure that the execution policy is set to unrestricted and the `LongPathsEnabled` registry key is set to `1`.
$ Set-ExecutionPolicy Unrestricted -Force
Set below value to 1 - 
$Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\FileSystem\LongPathsEnabled
How to Create a .exe file
To create an executable (.exe) file for the file finder application, follow these steps:

1. Activate the virtual Directory venv and then run the installer command

2. Run the following command in the terminal:
   
   pyinstaller --onefile <script_name>.py
  
3.    Replace `<script_name>` with the name of your Python script (`file_info_version_22`).
4. After successful execution, three files will be generated in the folder:
   - `.env`
   - `1_sql_file_info_create.sql`
   - `file_info_version_XYZ.exe`
5. Ensure that both the `.env` file and the executable file (`file_info_version_22.exe`) are in the same folder.
6. Run the `file_info_version_22.exe` file and follow the instructions displayed on the Command Line interface.
7. Wait for the search process to be completed.

Install PowerBI

To utilize PowerBI with the file finder application, follow these steps:

1. Install PowerBI from the Microsoft Store.
2. Install the PowerBI MySQL connector to establish a connection with the MySQL database.
Steps to Install the Connector:
• Open this link: https://dev.mysql.com/downloads/connector/net/
• Download the Connector for your OS.
• Open the installer file and finish the steps.
• Now, refresh PowerBI.
3. Click on File -> Get Data -> Get Data from Other Sources. (if you cant see it, scroll down and click “Get data to Get Started”). Type in the server name (Usually localhost:3306), the database name (lisney_files_info8), proceed, then type in the username (arungt) and password (fi!ef!ndgt!23) of the database. Proceed. The tables in the database will be shown, select all tables then load the data.

4. When opening a file in PowerBI, select `File` > `Security` > `Not Recommended` for enhanced security measures.
